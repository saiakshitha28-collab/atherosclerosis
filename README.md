# ❤️ Atherosclerosis Risk Prediction using Machine Learning

## 📌 Project Overview

Atherosclerosis Risk Prediction is a real-time web application developed using **Flask** and **Machine Learning**. The system predicts an individual's risk of developing atherosclerosis based on medical and lifestyle parameters, helping support early diagnosis and preventive healthcare.

This project was developed as a **College Mini Project** in the field of **Machine Learning and Healthcare Analytics**.

---

## 🎯 Objectives

- Predict the risk of atherosclerosis using machine learning.
- Analyze patient health data for early disease detection.
- Provide a simple web interface for healthcare prediction.
- Support preventive healthcare through data-driven insights.

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Pandas
- NumPy
- Scikit-learn
- Joblib / Pickle

---

## 🤖 Machine Learning

The application uses a trained machine learning model to classify whether a patient is at risk of developing atherosclerosis based on the provided health parameters.

## ▶️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Atherosclerosis_Risk_Prediction.git
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python app.py
```

### 4. Open in your browser

```
http://127.0.0.1:5000
```

---

## 📊 Application Workflow

1. Enter patient health details.
2. Click the **Predict** button.
3. The machine learning model processes the input.
4. The application displays the predicted risk.
